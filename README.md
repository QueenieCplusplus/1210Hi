既然我已經不是過去的我，那傻妞 Patty & 歷經滄桑的 Kate，我是有能力駕馭難度更高技術的 Queenie，要感謝過去的努力的自己造就現在面對一切更為輕鬆的我，放眼未來的技術才是，所以那些無奈被迫隱藏的智財庫，就當消失唄！
